# Trader Performance vs Market Sentiment Analysis

## 📌 Objective
Analyze how market sentiment (Fear vs Greed) impacts trader behavior and performance.

## 📊 Dataset
- Bitcoin Fear & Greed Index
- Hyperliquid Trader Data

## ⚙️ Steps Performed
- Data cleaning and preprocessing  
- Feature engineering (PnL, trade count)  
- Merging datasets on date  
- Visualization and analysis  

## 📈 Key Insights
- Traders earn higher profits during Extreme Greed  
- Trading activity increases during Fear periods  
- Market sentiment strongly influences trader behavior  

## 🚀 Strategy Recommendations
- During Fear → reduce trading and risk exposure  
- During Greed → trade actively but manage leverage  
- Use sentiment as a signal for decision-making
